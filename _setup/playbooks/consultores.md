# Playbook FlowOS Pro — Consultores e Prestadores de Serviço

**Para:** Consultores, coaches, contadores, advogados, designers, desenvolvedores, freelancers e qualquer prestador de serviço intelectual.

---

## Os 3 agentes que mudam o jogo para você primeiro

### 1. Orq. Propostas — Proposta que fecha sem precisar de reunião extra

Consultor que envia proposta profissional e clara fecha mais do que quem é melhor tecnicamente mas se comunica mal.

**Proposta de consultoria:**
```
/orquestra-propostas monta proposta de consultoria [área] para empresa de [porte]
Contexto: [o que você entendeu do problema deles]
Escopo: [o que você vai entregar]
Valor: R$[X]
```

**Proposta de projeto pontual:**
```
/orquestra-propostas monta proposta de projeto: [descrição]
Entregáveis: [lista]
Prazo: [X semanas]
Valor: R$[X]
```

**Follow-up de proposta sem resposta:**
```
/orquestra-propostas mandei proposta há 3 dias, cliente não respondeu. Monta follow-up sem pressão.
```

---

### 2. Alavanca — Transforma expertise em produto que escala

Todo consultor chega num teto de horas. A Alavanca transforma o que você sabe em produto que vende quando você está dormindo.

**Definindo o produto:**
```
/Alavanca sou consultor de [área]. Quero criar algo que escale meu conhecimento. Me ajuda a definir o produto certo.
```

**Estruturando uma mentoria:**
```
/Alavanca quero criar uma mentoria de [X meses] para [público]. Estrutura os módulos e o resultado esperado.
```

**Precificando corretamente:**
```
/Alavanca cobro R$300/hora em consultoria 1:1. Quanto cobro por uma mentoria em grupo de 8 semanas?
```

---

### 3. Vitrine — Relatório de resultado que justifica a renovação do contrato

Cliente renova quando percebe valor. O Vitrine faz esse valor ficar visível.

**Relatório mensal:**
```
/Vitrine monta relatório de resultado do mês para o cliente [nome]
O que foi feito: [lista]
Resultados: [métricas ou conquistas]
Próximos passos: [o que vem no próximo mês]
```

**Apresentação para renovação:**
```
/Vitrine vou apresentar resultados do trimestre para justificar renovação. Monta a estrutura da apresentação.
```

## Exemplos práticos por situação

**Dificuldade em precificar o serviço:**
```
/Farol cobro R$X/hora, tenho [N] horas disponíveis por mês. Meus custos fixos são R$Y. Preciso de quanto em contratos?
```

**Preparando call de diagnóstico:**
```
/Fisga vou ter call de diagnóstico com prospect amanhã. É uma [tipo de empresa]. Monta o roteiro de perguntas.
```

**Onboarding de cliente novo:**
```
/Trilha fechei consultoria com cliente [nome]. Serviço: [tipo]. Monta sequência de onboarding.
```

**Conteúdo de autoridade:**
```
/Holofote quero ser referência em [especialidade] no LinkedIn. Que tipo de conteúdo constrói isso?
```

**Quando o cliente pede desconto:**
```
/orquestra-propostas cliente pediu 20% de desconto na minha proposta de R$X. Como respondo valorizando o trabalho?
```

## Precificação — o erro mais comum do consultor

Se você está sempre ocupado mas não tem sobra no final do mês, o problema é precificação.

```
/Farol trabalho sozinho, tenho 20 horas disponíveis por semana para clientes. 
Custos fixos: R$[X]/mês. Quero R$[Y] líquido por mês. Quanto devo cobrar por hora?
```

O Farol calcula e te mostra o número real.
